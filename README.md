# Indiamart-Data-Extraction
# Automated IndiaMART Product Data Extraction

## Project Overview

This project automates product data extraction from IndiaMART using Python and Selenium.

## Features

* Automated browser navigation
* Product detail extraction
* Popup handling
* Excel data export
* Automated web scraping

## Technologies Used

* Python
* Selenium
* Pandas
* OpenPyXL

## Output

The extracted product details are stored in Excel format for analysis and reporting.

## How to Run

```bash
pip install -r requirements.txt
python main.py
```
